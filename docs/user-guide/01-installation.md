# 01 — 安装与启动

## 前置条件
- Windows 10/11（Mac/Linux 同样支持）
- Python 3.11 或更高版本
- Node.js 18 或更高版本

## 安装步骤

### 1. 获取安装包
解压 `yishao-agent.zip` 到你想要的目录，例如 `D:\yishao-agent\`。

### 2. 首次启动
双击 `start.bat`（Mac/Linux 用户运行 `start.sh`）。

首次启动时，脚本会自动安装依赖：
- Python 依赖（fastapi, uvicorn 等）
- 前端依赖（React, Vite 等）

等待 5-10 秒，浏览器会自动打开 `http://localhost:5173`。

### 3. 验证启动
看到左侧「一勺笔录」Logo 和导航菜单，说明启动成功。

## 停止应用
关闭命令行窗口即可停止服务。

## 常见错误

| 问题 | 解决 |
|------|------|
| "python 不是内部或外部命令" | 安装 Python 3.11+ 并添加到 PATH |
| "npm 不是内部或外部命令" | 安装 Node.js 18+ |
| 端口 8765/5173 被占用 | 关闭占用进程后重试 |
| 浏览器显示"无法连接" | 等待几秒让服务完全启动，然后刷新 |
