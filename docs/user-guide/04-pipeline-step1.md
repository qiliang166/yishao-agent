# 04 — 步骤 1：内容获取

## 功能位置
项目详情页 → **「步骤 1 · 内容获取」**

## 前置条件
- 已创建项目
- 已进入项目详情页

## 操作说明

### 三种输入方式

#### 方式一：粘贴链接（推荐）
1. 将视频链接（如抖音、B站、YouTube 链接）粘贴到输入框
2. 点击 **「开始下载」** 按钮
3. 等待下载完成，进度条实时显示状态
4. 下载完成后，字幕自动提取并显示在编辑区
5. 可手动编辑字幕文本

#### 方式二：直接输入文本
1. 在字幕编辑区直接粘贴或输入笔记文字
2. 点击 **「确认进入步骤 2」** 保存内容

#### 方式三：上传文本文件
1. 点击 **「上传文件」** 按钮
2. 选择 `.txt` 或 `.md` 文件
3. 文件内容自动加载到编辑区

### 进度状态

| 状态 | 说明 |
|------|------|
| 🔄 下载中 | 正在下载视频，进度条显示百分比 |
| 📝 提取字幕中 | 正在提取字幕文本 |
| ✅ 完成 | 字幕已提取，可编辑 |
| ❌ 失败 | 下载或提取失败，检查链接是否有效 |

## 预期结果
- 粘贴链接 → 自动下载视频 → 提取字幕 → 字幕文本显示在编辑区
- 编辑字幕 → 点击确认 → 步骤 1 标记完成，步骤 2 解锁

## 常见错误

| 问题 | 原因 | 解决 |
|------|------|------|
| 下载失败 | 链接无效或平台不支持 | 检查链接格式，确认平台在支持列表内 |
| 下载缓慢 | 网络问题 | 等待或尝试更换链接 |
| 字幕为空 | 视频无内嵌字幕且 Whisper 转录失败 | 手动输入文字内容 |
| yt-dlp 报错 | 缺少 VC++ 运行时 | 安装 Microsoft Visual C++ Redistributable |
