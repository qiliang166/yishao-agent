# 09 — 模板管理

## 功能位置
左侧导航 → **「模板管理」**

## 前置条件
- 应用已启动
- （可选）已准备好 .pptx 或 .docx 模板文件用于上传

## 操作说明

### 页面布局

```
┌──────────────────────────────────────────────┐
│  [PPT模板]  [SOP模板]                          │
├──────────────────────────────────────────────┤
│                                              │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│  │ 模板名称  │ │ 模板名称  │ │ 模板名称  │     │
│  │ PPT 徽章  │ │ PPT 徽章  │ │ PPT 徽章  │     │
│  │ 默认 ⭐   │ │          │ │          │     │
│  │ 关联Skill│ │ 关联Skill│ │ 关联Skill│     │
│  │ [编辑]   │ │ [编辑]   │ │ [编辑]   │     │
│  │ [删除]   │ │ [删除]   │ │ [删除]   │     │
│  └──────────┘ └──────────┘ └──────────┘     │
│                                              │
│  [+ 新建模板]                                 │
└──────────────────────────────────────────────┘
```

### 切换模板类型
- 点击顶部 **「PPT模板」** 标签查看 PPT 模板
- 点击 **「SOP模板」** 标签查看 SOP 模板

### 新建模板

1. 点击底部 **「+ 新建模板」** 按钮
2. 在弹出的对话框中填写：
   - **模板名称**：如「简约食谱模板」
   - **模板类型**：PPT 或 SOP
   - **关联 Skill**：从下拉菜单选择对应分类的提示词（生成时自动使用该 Skill 的输出格式）
3. （可选）配置品牌信息（JSON 格式）：
   ```json
   {
     "logo_position": "top-right",
     "copyright": "© 2026 一勺食谱课件Agent",
     "signature": ""
   }
   ```
   - `logo_position`：Logo 位置（留空则生成时不插入 Logo）
   - `copyright`：版权文字（留空则生成时不加版权）
   - `signature`：签名位置（留空则生成时不加签名）
4. 点击「保存」

### 上传模板文件
1. 新建或编辑模板时，点击文件上传区域
2. 选择 `.pptx`（PPT 模板）或 `.docx`（SOP 模板）文件
3. 文件上传到本地 `backend/data/templates/` 目录

### 设为默认
- 在模板卡片上点击 **「设为默认」**
- 默认模板在步骤 4 中自动选中
- 每种类型只能有一个默认模板

### 编辑 / 删除
- **编辑**：点击卡片上的「编辑」按钮 → 修改后保存
- **删除**：点击「删除」按钮 → 确认后移除

## 预期结果
- 新建 → 卡片网格中出现新卡片
- 设为默认 → 卡片显示 ⭐ 标识
- 上传模板文件 → 步骤 4 中可选择该模板

## 常见错误

| 问题 | 原因 | 解决 |
|------|------|------|
| 上传失败 | 文件格式不支持 | 确认上传 .pptx 或 .docx 文件 |
| 模板不显示 | 类型标签页未切换 | 点击对应标签页（PPT/SOP） |
| 生成时模板不生效 | 模板占位符不匹配 | 确认模板中的占位符格式为 {{变量名}} |
