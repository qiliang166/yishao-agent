# 03 — 项目管理

## 功能位置
首页（项目列表页）

## 前置条件
- 应用已启动
- 首次使用无前置要求

## 操作说明

### 新建项目
1. 点击右上角 **「+ 新建项目」** 按钮
2. 在弹出的输入框中输入食谱名称
3. 点击「确定」
4. 新项目出现在列表中，点击卡片进入项目

### 查看项目列表
首页自动加载所有项目，按最近更新时间排序。每个项目卡片显示：
- 项目名称
- 状态标签（草稿 / 进行中 / 已完成）
- 最后更新时间

### 删除项目
1. 在项目卡片右侧点击 **「删除」** 按钮
2. 确认对话框中选择「确认」
3. 项目从列表中移除

## 预期结果
- 新建后：列表中出现新卡片，自动跳转到项目详情
- 删除后：卡片从列表消失

## 常见错误
| 问题 | 原因 | 解决 |
|------|------|------|
| 新建后列表不刷新 | 网络延迟 | 手动刷新页面 (F5) |
| 删除失败弹窗 | API 请求失败 | 检查后端是否正常运行 |
