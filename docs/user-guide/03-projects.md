# 03 — 项目管理

## 功能位置
首页（项目列表页）

## 前置条件
- 应用已启动
- 首次使用无前置要求

## 操作说明

### 新建项目
1. 点击右上角 **「+ 新建项目」** 按钮
2. 在弹出的输入框中输入食谱名称
3. 点击「确定」
4. 新项目出现在列表中，点击卡片进入项目

### 查看项目列表
首页自动加载所有项目，按最近更新时间排序。每个项目卡片显示：
- 序号
- 项目名称
- 状态标签（草稿 / 已完成）
- 最后更新时间
- 操作按钮（删除 / 锁定）

列表底部显示项目总数，超过 20 条时自动分页。

### 状态切换
点击项目卡片上的状态标签可在「草稿」和「已完成」之间切换：
- **草稿**：默认状态
- **已完成**：标记为完成

项目详情页的状态栏也支持点击切换。

### 锁定项目
锁定后项目无法被删除：
- **项目详情页**：点击顶部信息栏的「🔓 解锁」按钮锁定，「🔒 已锁定」按钮解锁
- **项目列表页**：锁定项目显示 🔒 图标，不显示删除按钮

### 批量操作
勾选项目左侧的复选框后，全选栏出现批量操作按钮：
- **✅ 批量完成** — 标记所有选中项目为「已完成」
- **📝 批量草稿** — 标记所有选中项目为「草稿」
- **🔒 批量锁定** — 锁定所有选中项目
- **🔓 批量解锁** — 解锁所有选中项目
- **删除选中(N)** — 删除所有未锁定的选中项目，锁定的自动跳过

批量删除前会在确认框中显示：将删除多少个、多少个已锁定跳过及其名称。

### 删除项目
- **单个删除**：点击项目卡片上的「删除」按钮，确认后删除
- **批量删除**：勾选后在全选栏点击「删除选中(N)」
- 锁定项目不允许删除
- 删除项目会同时清理项目存储目录中的文件

### 搜索项目
顶部搜索框输入关键词，按名称过滤项目。

## 预期结果
- 新建后：列表中出现新卡片，自动跳转到项目详情
- 删除后：卡片从列表消失，存储目录同时清理
- 切换状态：标签文字即时更新
- 锁定后：删除按钮变为 🔒 图标

## 常见错误
| 问题 | 原因 | 解决 |
|------|------|------|
| 新建后列表不刷新 | 网络延迟 | 手动刷新页面 (F5) |
| 删除失败弹窗 | API 请求失败或项目已锁定 | 检查后端是否正常运行，确认项目未锁定 |
| 项目已锁定，无法删除 | 项目锁已开启 | 进入项目详情页解锁后再删除 |
