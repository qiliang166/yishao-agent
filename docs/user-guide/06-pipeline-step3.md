# 06 — 步骤 3：文档批量生成

## 功能位置
项目详情页 → **「步骤 3 · 文档生成」**

## 前置条件
- 步骤 2 已完成（笔记已整理确认）
- LLM 配置正常

## 操作说明

### 三列布局

步骤 3 以三列并排显示，每列对应一份独立文档：

| 列 | 文档类型 | 说明 |
|------|------|------|
| 左 | **道与术分析** | 食谱背后的原理与技巧分析 |
| 中 | **研习手册** | 结构化研习步骤手册 |
| 右 | **食谱 SOP** | 标准化操作流程文档 |

### 操作步骤（每列独立操作）

1. 点击文档列的 **「生成」** 按钮
2. 在弹出的 AI 协作编辑器中：
   - 选择该文档类型的 **提示词版本**
   - 选择 **LLM 模型**
   - 点击 **「生成初稿」**
3. 生成后可在编辑器中手动修改或使用 AI 协助（润色/扩写/精简）
4. 点击 **「确认」** 保存该文档

### 并发生成
- 三份文档可同时生成，互不影响
- 每个文档独立选择提示词和模型
- 可分别为三份文档选择不同的 LLM 提供商（如道与术用 DeepSeek，SOP 用通义千问）

### 步骤完成
- 三份文档全部确认后，步骤 3 标记完成
- 步骤 4 解锁

## 预期结果
- 每份文档独立生成，选择各自的提示词和模型
- 生成后可编辑修改
- 三份全部确认 → 可进入步骤 4

## 常见错误

| 问题 | 原因 | 解决 |
|------|------|------|
| 某文档生成失败 | 模型或提示词问题 | 更换模型或检查提示词格式 |
| 三个文档内容重复 | 提示词未区分文档类型 | 确认每个文档使用了对应分类的提示词 |
| 生成内容偏差 | 提示词与步骤 2 笔记内容不匹配 | 编辑提示词或手动调整笔记 |
