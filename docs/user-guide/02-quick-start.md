# 02 — 快速上手

## 你的第一个项目

### 步骤 1：创建项目
1. 在首页点击 **「+ 新建项目」** 按钮
2. 输入食谱名称，例如「红烧肉」
3. 点击确定

### 步骤 2：查看项目
创建后自动进入项目详情页（当前为占位页面，后续版本将显示完整的流水线工作台）。

### 步骤 3：返回首页
点击左侧导航 **「项目列表」** 即可返回。

## 当前可用功能
- 创建/查看/删除项目
- 配置 LLM API Key（设置页）
- 测试 LLM 连接
