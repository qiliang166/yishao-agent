# 05 — 步骤 2：笔记整理

## 功能位置
项目详情页 → **「步骤 2 · 笔记整理」**

## 前置条件
- 步骤 1 已完成（已有内容文本）
- 已配置至少一个 LLM 提供商
- （推荐）已创建「笔记整理」分类的提示词

## 操作说明

### AI 协作编辑器布局

```
┌─────────────────────────────┬──────────────────┐
│      文档编辑区（左侧）       │   AI 协助面板     │
│                             │   （右侧，可折叠）  │
│   - 自由编辑文字             │                  │
│   - 选中文字后可 AI 处理      │  - 选择提示词     │
│   - 支持 Ctrl+Z 撤销         │  - 选择模型      │
│                             │  - 点击操作按钮   │
│                             │  - 自定义指令输入 │
└─────────────────────────────┴──────────────────┘
```

### 生成初稿
1. 在右侧 AI 面板中选择 **提示词版本**（如「笔记整理 v1.0」）
2. 选择 **LLM 模型**（如 deepseek-chat）
3. 点击 **「生成初稿」**
4. AI 根据步骤 1 的内容和提示词生成笔记初稿
5. 初稿覆盖左侧编辑区，可继续手动修改

### AI 协助操作

| 操作 | 步骤 |
|------|------|
| **润色选中** | 选中文字 → 点击「润色选中」→ AI 仅改写选中部分 |
| **扩写** | 选中文字 → 点击「扩写」→ AI 将选中部分扩展 |
| **精简** | 选中文字 → 点击「精简」→ AI 将选中部分压缩 |
| **检查错别字** | 选中文字（或全选 Ctrl+A）→ 点击「检查错别字」→ AI 纠正错别字 |
| **自定义指令** | 选中文字（可选）→ 输入指令（如「改成更正式的语调」）→ 按回车 |

### 确认笔记
- 编辑满意后，点击底部 **「确认进入步骤 3」**
- 步骤 2 标记完成，步骤 3 解锁

## 预期结果
- 生成初稿 → 编辑器显示 AI 返回的内容
- 选中文字 + AI 操作 → 仅选中区域被替换
- 未选中 + 自定义指令 → AI 在光标位置插入新内容
- 确认后 → 内容保存为 step2，可进入步骤 3

## 常见错误

| 问题 | 原因 | 解决 |
|------|------|------|
| 生成失败 | LLM 配置错误或余额不足 | 检查设置页 LLM 配置，测试连接 |
| 生成内容为空 | 提示词 + 模型组合不匹配 | 尝试更换模型或提示词 |
| 选中替换失效 | 未正确选中文字 | 确认编辑器中有选中区域（高亮） |
| 生成超时 | 模型响应慢或内容过长 | 缩短步骤 1 的内容或更换更快的模型 |
