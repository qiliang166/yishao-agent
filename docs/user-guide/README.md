## 5 阶段一览

| # | 阶段 | 子标签 | 产出 |
|---|------|--------|------|
| 1 | 文案提取 | 视频提取 / 文字输入 / 文件上传 | 原始文本内容 |
| 2 | 教学文档 | SOP文案 / 道与术文案 / 研学手册文案 | 3 份 AI 生成文档 |
| 3 | 输出课件 | SOP课件 / 道术PPT / 研学PPT | .docx + 2 个 .pptx |
| 4 | 语音课件 | 口播文案 / 口播语音 | 口播稿 + .mp3 音频 |
| 5 | 输出列表 | (无) | 一键下载 .zip |

### 数据流向

- Stage 1 输出 → Stage 2 输入
- Stage 2 三路输出 → Stage 3 对应三路输入
- Stage 2 研学手册文案 → Stage 4 口播文案 → 口播语音
- Stage 3 + Stage 4 所有产出 → Stage 5 汇总下载
