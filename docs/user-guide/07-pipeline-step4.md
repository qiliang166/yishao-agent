# 07 — 步骤 4：多格式输出

## 功能位置
项目详情页 → **「步骤 4 · 输出与下载」**

## 前置条件
- 步骤 3 已完成（至少一份文档生成）
- PPT/SOP 模板已配置（可选，无模板则使用默认样式）

## 操作说明

### PPT 生成

1. 在 PPT 区域输入或粘贴要生成的内容（通常来自道与术分析或研习手册）
2. 从下拉菜单选择 **PPT 模板**（需先在模板管理中上传）
3. 填写品牌信息（可选）：
   - **版权文字**：如「© 2026 一勺笔录」
   - **签名位置**：如「—— 张三」
   - **Logo 路径**：留白则生成时该位置空置
4. 点击 **「生成 PPT」**
5. 生成完成后，点击 **「下载 .pptx」** 保存到本地

### SOP 导出

1. 在 SOP 区域输入或粘贴内容（通常来自食谱 SOP 文档）
2. 选择 **SOP 模版**（可选）
3. 填写品牌信息（可选，同上）
4. 点击 **「导出 SOP」**
5. 生成完成后，点击 **「下载 .docx」** 保存到本地

### 口播稿 + TTS 语音合成

1. 在口播稿区域，点击 **「从道与术生成」** 或 **「从研习手册生成」**
2. AI 根据选中的文档生成适合朗读的口播稿
3. 手动编辑口播稿（调整语气、断句、时长等）
4. 点击 **「语音合成」**
5. 等待 TTS 处理（长文本自动分段合成）
6. 合成完成后：
   - 点击播放按钮 **试听**
   - 点击 **「下载音频」** 保存 .mp3 文件

### 一键导出全部
- 底部状态栏的 **「全部导出 .zip」** 按钮可将 PPT + SOP + 音频打包下载

## 预期结果
- PPT 生成 → 下载 .pptx 文件可正常打开
- SOP 导出 → 下载 .docx 文件可正常打开
- TTS 合成 → 可播放和下载 .mp3 音频
- 一键导出 → 下载包含所有文件的 .zip

## 常见错误

| 问题 | 原因 | 解决 |
|------|------|------|
| PPT 生成失败 | python-pptx 模板解析错误 | 检查模板文件格式，或使用默认模板 |
| SOP 导出失败 | python-docx 写入错误 | 检查内容格式，避免特殊字符 |
| TTS 合成失败 | DashScope API Key 未配置或余额不足 | 检查设置页 TTS 配置 |
| 下载链接失效 | 文件已被清理 | 重新生成后下载 |
| 音频无声 | TTS 合成参数问题 | 调整音量（volume）参数，默认 50 |
