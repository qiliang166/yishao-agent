# 08 — 提示词管理

## 功能位置
左侧导航 → **「提示词管理」**

## 前置条件
- 应用已启动

## 操作说明

### 页面布局

### 新建提示词

1. 点击左下角 **「+ 新建」** 按钮
2. 在右侧编辑区填入：
   - **名称**：如「默认笔记整理」
   - **分类**：从下拉菜单选择（笔记整理 / 道与术 / 研习 / SOP / 口播稿 / PPT Skill）
   - **系统 Prompt**：AI 的角色设定和指令
   - **Skill 模板**：输出格式规范（如标题层级、分段结构等）
3. 点击 **「保存」**
4. 新提示词出现在左侧列表中

### 编辑提示词（自动版本管理）

1. 在左侧列表点击选中一个提示词
2. 修改系统 Prompt 或 Skill 模板
3. 在 **「变更说明」** 输入框填写本次修改内容（如「增加了食材用量格式要求」）
4. 点击 **「保存」**
5. 系统自动创建新版本：v1.0 → v1.1 → v1.2 ...

### 版本管理

| 操作 | 说明 |
|------|------|
| **查看版本历史** | 选中提示词后，底部显示版本列表（版本号 + 时间 + 变更说明） |
| **版本回滚** | 点击某版本的「回滚」按钮 → 当前内容替换为该版本 |
| **版本对比** | 勾选两个版本 → 点击「对比」→ 弹出 diff 模态框，增删内容高亮显示 |

### 设为默认
- 在提示词列表中找到目标提示词，点击 **「设为默认」**
- 设为默认后，AI 协作编辑器中该分类将自动选中此提示词

### 导入 / 导出

- **导出**：点击左下角「导出」按钮 → 下载 `prompts-export.json`（不含 API Key）
- **导入**：点击左下角「导入」按钮 → 选择之前导出的 JSON 文件 → 提示词批量导入

## 预期结果
- 新建 → 左侧列表出现新条目
- 编辑保存 → 自动创建新版本
- 回滚 → 当前内容替换为选中的历史版本
- 对比 → 显示两个版本之间的差异
- 导出 → 下载 JSON 文件；导入 → 恢复提示词

## 常见错误

| 问题 | 原因 | 解决 |
|------|------|------|
| 保存无反应 | 名称或分类为空 | 填写必填字段 |
| 版本对比为空 | 两个版本内容相同 | 选择不同版本对比 |
| 导入失败 | JSON 格式不正确 | 确认文件为系统导出的原始格式 |
