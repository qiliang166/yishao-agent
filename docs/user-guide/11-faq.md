# 11 — 常见问题（FAQ）

## 安装与启动

### Q: 双击 start.bat 后窗口闪退？
A: 可能缺少 Python 或 Node.js。打开命令行（cmd），分别运行 `python --version` 和 `node --version` 确认已安装。

### Q: 浏览器打开后显示"无法连接"？
A: 后端服务可能还在启动中。等待 5-10 秒后刷新页面。如果仍然无法连接，检查命令行窗口是否有错误信息。

### Q: 端口 8765 或 5173 被占用怎么办？
A: 关闭占用端口的程序。可以在命令行运行 `netstat -ano | findstr 8765` 查找占用进程 PID，然后在任务管理器中结束该进程。

## LLM 配置

### Q: 支持哪些大模型？
A: 任何兼容 OpenAI API 格式的提供商均可使用，包括 DeepSeek、通义千问、Kimi、智谱 GLM 等。

### Q: 测试连接成功但生成失败？
A: 可能原因：(1) API 余额不足 (2) 模型名称拼写错误（注意大小写）(3) 提示词过长超出上下文限制。

### Q: API Key 安全吗？
A: API Key 存储在本地 SQLite 数据库中，不会上传到任何服务器。前端显示时默认隐藏。导出数据时不包含 API Key。

## 视频下载

### Q: 支持哪些视频平台？
A: yt-dlp 支持 1000+ 平台，常见的有：抖音、B站（Bilibili）、YouTube、西瓜视频、小红书等。

### Q: 下载视频后没有字幕？
A: 优先提取内嵌字幕；如果视频无内嵌字幕，系统会使用 Whisper 进行本地语音识别。此过程较慢，需耐心等待。

### Q: 下载速度很慢？
A: 视频下载速度取决于网络环境和平台限速，属于正常现象。

## 提示词

### Q: 如何写出好的提示词？
A: 建议：(1) 明确角色设定（"你是一位专业的食谱分析师"）(2) 指定输出格式（标题层级、段落结构）(3) 给出具体约束（字数、语气）。可在 Skill 模板中定义输出格式。

### Q: 可以分享提示词吗？
A: 使用导出功能可将所有提示词导出为 JSON 文件，分享给他人导入使用。导出不包含 API Key。

## 数据存储

### Q: 数据存在哪里？
A: 所有数据存储在 `backend/data/` 目录下，包括 SQLite 数据库、导出的文件、音频文件等。可在设置页查看占用空间。

### Q: 如何备份数据？
A: 后端启动时自动备份数据库到 `backend/data/backups/`，保留最近 7 天的备份。也可手动复制整个 `backend/data/` 目录。

### Q: 如何迁移到另一台电脑？
A: 复制整个 `backend/data/` 目录到新电脑的对应位置即可。提示词可通过导入/导出 JSON 迁移。
