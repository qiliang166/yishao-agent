## 视频播放校验修复

### 问题
视频校验弹窗中视频无法播放，`/api/video/file` 端点返回 403。

### 根因
Windows 路径大小写不敏感但 Python `str.startswith` 大小写敏感：
- `__file__` 返回 `D:\YISHAOAGENT\...` (大写 D)
- URL 参数解码后得到 `d:\YISHAOAGENT\...` (小写 d)
- `os.path.normpath()` 不改变盘符大小写
- 导致 `startswith` 检查失败

### 修复
- 使用 `os.path.normcase()` 统一转为小写后再比较
- 扩展允许路径范围：增加 `_get_global_save_path()` 根目录
- 后端: `backend/app.py` — `api_video_file` 端点

### 视频校验弹窗增强
- 标题栏可拖曳移动窗口位置
- 右下角可拖曳调整窗口大小

## PPT 模板生成规则系统

### 背景
将栏目配置（col4/col5）从「两个 textarea」升级为「结构化 PPT SKILL 规则系统」，作为该栏目下所有模板的**分析框架和生成约束**。

### 设计参照
参照 [guizang-ppt-skill](https://github.com/op7418/guizang-ppt-skill) 的多维结构。

### rules JSON 字段（8 字段）
| # | 字段 | 说明 |
|---|------|------|
| 1 | `design_rules` | 5套主题色预设 + 色彩纪律 + 字重阶梯 + 页面比例 + 品牌区 |
| 2 | `layout_types` | 版式类型定义（col4=6种, col5=7种），每种含 zones + description |
| 3 | `components` | 原子组件规范：字体阶梯 / 网格 / callouts(4种) / stats(3种) / pipeline(3种) |
| 4 | `content_spec` | Markdown 内容结构模板 |
| 5 | `image_rules` | 配图类型 + 生图提示词 + 放置规则 |
| 6 | `clarification_questions` | 6问清单，Agent 生成前引导用户 |
| 7 | `design_principles` | 6条设计铁律（克制/结构/字体/图片/节奏/术语） |
| 8 | `page_rhythm` | 页面节奏规则 + P0 违规定义 |
| 9 | `checklist` | P0/P1/P2/P3 分级质量检查清单 |
| 10 | `analysis_rules` | 模板解析提示词（分析上传 .pptx → 生成 prompt+skill） |

### 架构
```
栏目配置（规则框架）          模板管理（具体实例）
┌─────────────────┐         ┌─────────────────┐
│ 设计纪律         │         │ 上传 .pptx      │
│ 版式类型定义     │  ──→   │ 自动解析结构     │
│ 配色规则         │  框架   │ AI 在框架内补全  │
│ 排版/组件规范    │  约束   │ → prompt+skill  │
│ 配图/检查清单    │         │ 不偏离框架      │
└─────────────────┘         └─────────────────┘
```
