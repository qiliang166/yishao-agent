## 视频播放校验修复

### 问题
视频校验弹窗中视频无法播放，`/api/video/file` 端点返回 403。

### 根因
Windows 路径大小写不敏感但 Python `str.startswith` 大小写敏感：
- `__file__` 返回 `D:\YISHAOAGENT\...` (大写 D)
- URL 参数解码后得到 `d:\YISHAOAGENT\...` (小写 d)
- `os.path.normpath()` 不改变盘符大小写
- 导致 `startswith` 检查失败

### 修复
- 使用 `os.path.normcase()` 统一转为小写后再比较
- 扩展允许路径范围：增加 `_get_global_save_path()` 根目录
- 后端: `backend/app.py` — `api_video_file` 端点

### 视频校验弹窗增强
