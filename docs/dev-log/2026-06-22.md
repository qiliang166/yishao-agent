## 遇到的问题
- task-brief 和 review-package 脚本 exit code 49（Python/Windows 兼容问题）→ 手动创建 brief 和 diff 文件
- start.bat 初始使用硬编码绝对路径 → 修复为 %~dp0 相对路径
- update_project 对不存在的项目返回 500 → 添加 None 检查返回 404
- init_db 缺少 try/finally → 添加连接释放保护
- HomePage 缺少错误处理 → 重构使用 api.ts + try/catch
- yt-dlp 在 Windows 上需要 Microsoft Visual C++ 运行时 → 文档中标注前置条件
- TTS 音频文件在 Windows 上的路径兼容 → 使用 os.path.join
- NSIS makensis 并非所有开发者都安装 → 提供 build.bat/build.ps1 两种构建方式 + 便携版 ZIP 兜底
