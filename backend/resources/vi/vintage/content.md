# 内容页 — 通用内容展示，Notion 卡片骨架：彩色描边 + 可选图标 + 文字驱动

> 内容页是演示的主体，Notion 风格以卡片边框本身作为彩色描边（1px solid chart_color）替代 business 的左侧 4px 粗色条，图标改为可选。

---

## 布局规则

- **背景**: `{{background}}`
- **布局**: 按布局选择决策树（10 种布局）
- **顶部色条**: 无（Notion 无全宽顶部 accent 色条）
- **标题**: 36-44px, `{{primary}}`,'Playfair Display', font-weight:700
- **标题下方**: accent 短线 32×2px `{{accent}}`（可选，Type C 极简模式可省略）
- **内容区**: 占画布 55-70%
- **页码**: 右下角 dot(4px `{{primary}}`) + 当前页/总页数，13px Lora

## 图层结构（3 层）

1. **背景层**: `{{background}}` 铺满（纯色，无几何装饰）
2. **结构层**: 卡片容器 + 网格排列（按 10 种布局）
3. **内容层**: 文字 + 可选图标 + 图表

> Notion 仅 3 层。无 business 的"装饰层"（几何图形）和"标识层"（全宽顶部色条）。

## 卡片系统（Notion 骨架）

每张卡片遵循以下结构：
```
┌─ 卡片容器（card_bg + 6px 圆角 + 1px solid chart_color 彩色描边 + 无阴影）──────┐
│                                                                                 │
│ [可选] SVG 图标（16px，opacity:0.6）+ 标题（primary 色）                          │
│ 正文（body 字号 15-17px，line-height: 1.65-1.8，text 色）                        │
│ [可选] 图表区 / 底部标签                                                          │
└─────────────────────────────────────────────────────────────────────────────────┘
```

| 属性 | 值 |
|------|-----|
| 背景 | `{{card_bg}}` |
| 圆角 | 6px |
| 阴影 | none |
| 边框 | `1px solid chart_color`（彩色描边，边框即装饰），hero/step_N 使用默认 `1px solid rgba(55,53,47, 0.09)` |
| 卡片间距 | 28px |
| 标题字号 | 18-22px, `{{primary}}`,'Playfair Display', font-weight:600, 上间距 20px |
| 正文字号 | 15-17px, `{{text}}`,'Playfair Display', line-height:1.65-1.8, 上间距 8px |
| 图标 | 16px, outline 风格, 跟随卡片 chart_color, opacity:0.6, **可选** |

## 卡片彩色描边轮换规则

- N 张卡片 → 使用 chart_colors[0..N-1]，每卡一个不同色边框
- hero 卡无彩色描边（使用默认灰色边框 `1px solid rgba(55,53,47, 0.09)`）
- step_N 卡无彩色描边（使用默认灰色边框）
- 禁止所有卡片同一颜色边框
- 禁止 accent 色 `{{accent}}` 用于卡片彩色描边（accent 是页面级装饰色）

## 图标使用规则

- **文字优先**：标题和正文本身应足以传达信息
- **可选添加**：同一页面最多 2-3 张卡片使用图标
- **纯文字卡片**是正确的设计选择，不是遗漏

## 跨页一致性

所有内容页固定元素：
- 标题位置（同 left/同 top）
- 标题短线（同色同尺寸 32×2px，可选）
- 页码标记（同位置同风格）
- 字体族（全 deck 统一 'Playfair Display' 栈）
- 卡片间距（同 gap 值 28px）
- 卡片圆角（同 6px）

## 视觉丰富度

- SVG 图标元素 ≥2 个（非必须，但至少 2 个页面使用）
- ≥3 种不同 chart_color 出现在页面
- 每张卡片：标题 + 正文（图标和图表的出现取决于信息需要）
- 3 层结构完整
- 无阴影、无渐变、无全宽顶部色条
- 卡片边框本身即是彩色装饰，无附加色条
