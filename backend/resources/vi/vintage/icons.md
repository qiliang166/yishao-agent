# 图标规范 — SVG 图标风格、尺寸与颜色规则

> 图标在 Notion 风格中为**可选元素**，优先以文字和排版驱动信息传达。仅在需要快速视觉识别时添加图标。

---

## 图标使用原则

- **文字优先**：标题和正文本身应足以传达信息，图标仅作为辅助视觉锚点
- **少即是多**：同一页面最多 2-3 张卡片使用图标，其余卡片纯文字
- **尺寸克制**：图标只做点缀，不与文字争夺注意力
- **封面/总结**：不使用图标，纯排版驱动

## 图标规格

| 属性 | 值 |
|------|-----|
| 风格 | outline（描线图标，圆角端点） |
| 描边宽度 | 1.5px |
| card 图标尺寸 | 16px |
| hero 图标尺寸 | 28px |
| metric 图标尺寸 | 14px |
| 颜色规则 | 跟随所在卡片的 chart_color, opacity: 0.6 |

## 图标集（10 个，精简集）

| 名称 | 用途 |
|------|------|
| clock | 时间、进度、效率 |
| user | 用户、团队、角色 |
| chart | 数据、趋势、分析 |
| shield | 安全、保障、防护 |
| star | 评分、推荐、亮点 |
| check | 确认、通过、完成 |
| target | 目标、指标、定位 |
| book | 知识、文档、学习 |
| compare | 对比、差异、权衡 |
| flag | 里程碑、标记、起点 |

> 相比 business 风格移除了 `zap`（性能）和 `grid`（网格），Notion 风格更偏文档化，不需要速度和模块化图标。
