# 章节分隔页 — 纯白背景，居中章节编号+标题，无装饰，用于内容板块之间的视觉分隔

> 章节分隔页以极简方式呈现：仅大号编号 + 标题居中，无 accent 下划线，无顶部色条，大量留白。

---

## 继承规则

章节分隔页继承 [总结页](summary.md) 的纯白背景和零装饰规则，并叠加以下专属规范。

## 页面级 Override

| 属性 | 值 |
|------|-----|
| heading_scale | 1.15 |
| 默认布局 | full_bleed（全屏居中） |
| 文字对齐 | 居中 |
| 卡片容器 | 禁止使用卡片 |
| 顶部色条 | 无 |
| 标题短线 | 无 |

---

## 章节分隔规则

### 章节编号
- 纯文字编号（如 "03" / "Part 2" / "第三章"）
- 字号: 16-20px
- 颜色: `{{secondary}}`, opacity 0.6
- 字体: 'Playfair Display', weight 400
- 无圆圈、无方块、无几何包裹

### 章节标题
- 字号: 42-52px
- 颜色: `{{primary}}`
- 字体: 'Playfair Display', weight 700
- **下方无 accent 短线**（与 business 风格的关键差异）

### 装饰
- 零装饰。无顶部色条、无标题短线、无几何图形、无阴影、无渐变

---

## 内容结构

- 章节编号: "Part 2" / "03" / "第三章"
- 章节标题: 简洁有力（≤12 字）
- 可选副标题: 章节概述（≤30 字, 15px, `{{secondary}}`, opacity 0.6）
- 无页码标记

## 与 business 风格的关键差异

| 特征 | business | notion |
|------|----------|--------|
| 标题下方 | accent 下划线 | 无 |
| 顶部色条 | 有 | 无 |
| 章节编号 | 彩色 accent | `{{secondary}}` 灰色 |
| 背景 | card_bg 浅灰 | background 纯白 |
