# 标题 — 文档主标题行，card_bg 纯色背景 + accent 下划线

> 标题是文档的第一视觉锚点，必须醒目且可识别。所有文档模板均包含此段落。

---

## 结构规则

- **列数**: 全宽合并（colspan = 文档总列数）
- **背景**: `{{card_bg}}`
- **文字色**: `{{primary}}`
- **字号**: 20-24px, font-weight: 700
- **字体**: 'Playfair Display' / heading font
- **字距**: letter-spacing: 2-3px
- **内边距**: 14-18px
- **底部装饰**: 2px solid `{{accent}}`（分隔标题与正文）

## 文字内容

| 字段 | 说明 |
|------|------|
| text | 主标题文字（如 "作业手册标准文档"） |

## 使用场景

- 所有文档模板必有段落
- 标题文字由模板的 `title` 字段指定
