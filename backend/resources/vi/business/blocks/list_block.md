# 列表块 — 编号或项目符号列表，用于决议、要点、清单

> 列表块展示有序或无序的条列式内容。

---

## 结构规则

- **列数**: 全宽合并（colspan = 文档总列数）
- **列表项**: 左侧编号/符号 + 右侧文字
  - 编号: `{{primary}}`, font-weight: 600, 14px, 宽度 36px, 右对齐
  - 文字: `{{text}}`, 14px, 左对齐
- **行距**: 每项 10px padding, line-height: 1.6
- **项间**: 无分隔线或 `1px solid rgba({{text_rgb}}, 0.05)` 浅分隔
- **背景**: `{{background}}`
- **顶部**: `1px solid rgba({{text_rgb}}, 0.1)` 分隔线

## 编号样式

| 样式 | 示例 | 使用场景 |
|------|------|---------|
| ordered | 1. 2. 3. | 决议、步骤、优先级 |
| bullet | · · · | 要点清单、注意事项 |
| checkbox | ☐ ☐ ☐ | 待办事项、检查清单 |

## 模板字段

| 字段 | 类型 | 说明 |
|------|------|------|
| items | array | 列表项文字 |
| style | string | `ordered` / `bullet` / `checkbox` |
| indent | boolean | 是否缩进（默认 false） |

## 使用场景

- 会议记录：决议事项
- 标书：资质清单、项目清单
- 实验报告：注意事项
