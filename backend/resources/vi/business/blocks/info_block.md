# 信息块 — 键值对元数据表，标签列 + 值列的两栏信息展示

> 信息块用于展示结构化元数据（如食谱名称、参会人员、样本编号等），可附带图片占位区。

---

## 结构规则

- **列数**: 由模板指定（通常 3-6 列），左侧标签列 + 右侧值列
- **标签列**: `{{card_bg}}` 背景, `{{primary}}` 文字色, font-weight: 600, 居中
- **值列**: `{{background}}` 背景, `{{text}}` 文字色, 左对齐
- **行数**: 由模板数据决定
- **字号**: 13-14px
- **行高**: 10px padding

## 可选：图片占位区

- 值列右侧可 `rowspan` 合并多行作为图片占位
- 占位区: `{{card_bg}}` 背景, 虚线边框 `rgba({{accent_rgb}}, 0.35)`, 圆角 8px
- 内含 SVG 图片图标, stroke: `rgba({{accent_rgb}}, 0.4)`
- 标注文字: `rgba({{text_rgb}}, 0.45)`, 12px

## 模板字段

| 字段 | 类型 | 说明 |
|------|------|------|
| rows | array | 每项的 label + value |
| image_placeholder | boolean | 是否显示图片占位区 |
| image_rowspan | number | 图片占位合并行数 |
| image_colspan | number | 图片占位列数 |

## 使用场景

- 作业手册：食谱基本信息（4 行 + 图片占位）
- 会议记录：参会人员、时间地点
- 实验报告：样本编号、实验条件
- 标书：项目基本信息
