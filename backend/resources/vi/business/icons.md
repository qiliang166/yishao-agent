# 图标规范 — SVG 图标风格、尺寸与颜色规则

> 所有图标使用统一的描线风格（outline），圆角端点。

---

## 图标规格

| 属性 | 值 |
|------|-----|
| 风格 | outline（描线图标，圆角端点） |
| 描边宽度 | 1.5px |
| card 图标尺寸 | 20px |
| hero 图标尺寸 | 36px |
| metric 图标尺寸 | 16px |
| 颜色规则 | 跟随所在卡片的 chart_color, opacity: 0.7 |

## 图标集（12 个）

| 名称 | 用途 |
|------|------|
| clock | 时间、进度、效率 |
| user | 用户、团队、角色 |
| chart | 数据、趋势、分析 |
| shield | 安全、保障、防护 |
| star | 评分、推荐、亮点 |
| check | 确认、通过、完成 |
| target | 目标、指标、定位 |
| zap | 性能、加速、快速 |
| book | 知识、文档、学习 |
| grid | 网格、分类、模块 |
| compare | 对比、差异、权衡 |
| flag | 里程碑、标记、起点 |
