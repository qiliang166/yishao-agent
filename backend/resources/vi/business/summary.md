# 总结页 — 结尾收束，核心要点回顾与行动号召，深色背景与封面形成书挡效应

> 总结页与封面页形成"书挡效应"——同色系深色背景，首尾呼应。

---

## 布局规则

- **布局**: full_bleed（全屏垂直居中）
- **背景**: `linear-gradient(135deg, {{primary}} 0%, {{secondary}} 100%)`（与封面同款 hero_bg 渐变）
- **装饰**: 与封面同款装饰（光晕、大圆、几何图形），保持一致
- **文字色**: 全部白色 `#ffffff`

## Slide Type Override

| 属性 | 值 |
|------|-----|
| background | hero_bg 渐变 |
| text | `#ffffff` |
| heading_scale | 1.1（标题略大于普通内容页） |
| 布局 | full_bleed |

## 图层结构（5 层）

1. **背景层**: hero_bg 渐变铺满
2. **装饰层**: 与封面同款的大圆 + 光晕（opacity 0.03-0.08）
3. **结构层**: 无卡片容器（文字直接定位）
4. **内容层**: 标题 + 要点列表 + 联系方式
5. **标识层**: 顶部 accent 色条

## 书挡效应检查清单

与封面页的对应关系：
- [ ] 同色系深色背景（hero_bg 渐变）
- [ ] 同款装饰（光晕、大圆、几何图形）
- [ ] 白色文字
- [ ] 相同字体与字重
- [ ] 禁止卡片容器包裹内容

## 内容结构

- 简短总结（2-4 个关键要点）
- 每个要点: 图标 + 一句话
- 底部: 联系方式（邮箱 + 网址）或感谢语
- 页码: 最后一页（accent dot + 页号）

## 页脚呼应

首页 hero（深蓝底 + 底部黄铜粗线）与总结页（深蓝底 + 顶部黄铜粗线）形成视觉闭环。总结页作为文档的正式收束，深蓝色背景呼应封面权威感。

## 版权规范

总结页 / Thank-you 页必须在联系方式下方附加版权声明：
- 格式: `© YYYY Company Name. All rights reserved.`
- 字号: 11-12px
- opacity: 0.4-0.5
- 不抢主视觉但提升法律合规与正式感
