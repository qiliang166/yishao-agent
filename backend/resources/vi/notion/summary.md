# 总结页 — 结尾收束，纯白背景 + 文字居中，与封面形成零装饰书挡效应

> 总结页与封面页形成"书挡效应"——同为纯白背景、零装饰、纯排版驱动，首尾呼应。

---

## 布局规则

- **布局**: full_bleed（全屏垂直居中）
- **背景**: 纯色 `{{background}}`（纯白，与封面相同）
- **装饰**: 零装饰。无顶部色条、无标题短线、无几何图形、无阴影、无渐变
- **文字色**: 全部 `{{primary}}` / `{{text}}`
- **禁止卡片容器包裹内容** — 文字直接定位在画布上

## Slide Type Override

| 属性 | 值 |
|------|-----|
| background | 纯色 `{{background}}`（纯白，非 card_bg） |
| text | `{{primary}}` |
| heading_scale | 1.1 |
| 布局 | full_bleed |

## 图层结构（2 层）

1. **背景层**: 纯色 `{{background}}` 铺满
2. **内容层**: 标题 + 要点列表 + 联系方式

> Notion 总结仅 2 层。无结构层（无卡片），无标识层（无顶部色条、无标题短线）。

## 书挡效应检查清单

与封面页的对应关系：
- [ ] 同色纯白背景（`{{background}}`，非 card_bg）
- [ ] 零装饰（无顶部色条、无标题短线、无几何图形、无阴影、无渐变）
- [ ] 相同字体与字重
- [ ] 大量留白
- [ ] 禁止卡片容器包裹内容

## 内容结构

- 标题（如 "总结" / "关键要点"，32-40px, `{{primary}}`, weight 700）
- 简短总结（2-4 个关键要点，每点一行，15-17px, `{{text}}`）
- 底部: 联系方式（邮箱 + 网址）或感谢语（13px, `{{secondary}}`, opacity 0.5）
- 页码: 最后一页（dot(4px `{{primary}}`) + 页号）

## 版权规范

总结页 / Thank-you 页必须在联系方式下方附加版权声明：
- 格式: `(c) YYYY Company Name. All rights reserved.`
- 字号: 11-12px
- opacity: 0.4-0.5
- 颜色: `{{secondary}}`

## 与 business 风格的关键差异

| 特征 | business | notion |
|------|----------|--------|
| 顶部色条 | 有（全宽 accent） | 无 |
| 标题短线 | 有 | 无 |
| 背景 | card_bg 浅灰 | background 纯白 |
| 图层 | 4 层 | 2 层 |
| 卡片容器 | 无 | 无（相同） |
