# 结尾 — 文档收束区域，承载结论/签名/落款/日期

> 结尾是文档正文的正式收束，可选段落。

---

## 结构规则

- **列数**: 全宽合并（colspan = 文档总列数）或左右分栏（签名左 + 日期右）
- **背景**: `{{card_bg}}` 或 `{{background}}`（模板指定）
- **文字色**: `{{text}}`
- **字号**: 13px
- **内边距**: 12px 16px
- **顶部**: `2px solid rgba({{accent_rgb}}, 0.15)` 分隔线（呼应标题底部 accent 线，形成书挡效应）

## 典型内容

| 类型 | 示例 |
|------|------|
| 签名区 | "签字：__________" 左对齐 |
| 日期 | "日期：__________" 右对齐 |
| 结论 | "综上所述，..." 居中 |
| 落款 | 公司名称 + 日期 |

## 模板字段

| 字段 | 类型 | 说明 |
|------|------|------|
| content | string | 结尾文字 |
| layout | string | `center` / `split`（签名左+日期右） |

## 使用场景

- 会议记录：签名 + 日期
- 标书：盖章 + 落款
- 实验报告：结论 + 签名
- 作业手册：省略（直接用页脚版权收尾）
