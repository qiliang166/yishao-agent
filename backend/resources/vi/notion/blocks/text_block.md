# 文字块 — 自由段落文字，用于叙述性内容、分析说明、背景介绍

> 文字块承载非表格化的叙述性内容，全宽或指定列宽。

---

## 结构规则

- **列数**: 全宽合并（colspan = 文档总列数）
- **背景**: `{{background}}` 或 `{{card_bg}}`（模板指定）
- **文字色**: `{{text}}`
- **字号**: 13-15px, line-height: 1.8
- **内边距**: 12px 16px
- **首行缩进**: 可选（中文文档建议 2em）
- **小标题**: 15px, font-weight: 600, color: `{{primary}}`
- **顶部**: `1px solid rgba({{text_rgb}}, 0.1)` 分隔线

## 模板字段

| 字段 | 类型 | 说明 |
|------|------|------|
| text | string | 文字内容 |
| bg | string | 背景色（可选，默认 `{{background}}`） |
| indent | boolean | 首行缩进（可选，默认 false） |

## 使用场景

- 所有文档：公司简介、背景说明、分析总结
- 标书：公司资质说明
- 实验报告：数据分析、结论讨论
- 财务报告：经营情况说明
