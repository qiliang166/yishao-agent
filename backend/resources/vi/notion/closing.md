# 结尾页 — 演示文稿正式结束页，纯白背景零装饰，与封面形成首尾呼应

> 结尾页继承 [总结页](summary.md) 的纯白背景和零装饰规则，叠加感谢语和行动号召。

---

## 继承规则

结尾页继承总结页的纯白背景 + 零装饰规则，并叠加以下专属规范。

## 页面级 Override

| 属性 | 值 |
|------|-----|
| heading_scale | 1.1 |
| 默认布局 | full_bleed（全屏居中） |
| 卡片容器 | 禁止使用卡片 |
| 顶部色条 | 无 |
| 标题短线 | 无 |
| 装饰 | 零装饰 |

---

## 结尾展示规则

### 感谢语
- 主标题: "感谢聆听" / "Thank You" / "谢谢"
- 字号: 36-44px
- 字体: Inter, weight 700
- 颜色: `{{primary}}`
- 无下划线、无装饰

### 行动号召 (CTA)
- 核心行动引导（如 "扫码了解更多"）
- 字号: 15px
- 颜色: `{{accent}}`

### 联系方式
- 邮箱、网址、电话、地址
- 字号: 13px
- 颜色: `{{secondary}}`, opacity 0.5-0.6
- 排列: 横排或竖排，居中对齐

### 版权声明
- 格式: (c) YYYY Company Name. All rights reserved.
- 字号: 11-12px
- 颜色: `{{secondary}}`, opacity 0.35-0.45

---

## 内容结构

- 感谢语（必有）
- 行动号召（可选）
- 联系方式（可选）
- 版权声明（必有）

## 与 business 风格的关键差异

| 特征 | business | notion |
|------|----------|--------|
| 顶部色条 | 有 | 无 |
| 标题短线 | 有 | 无 |
| 背景 | card_bg 浅灰 | background 纯白 |
| 装饰 | 有几何元素 | 零装饰 |
