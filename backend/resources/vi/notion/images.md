# 配图规范 — 配图/插图的使用时机与 SVG 排版规范

> Notion 风格以文字排版为主，配图/插图为辅。仅在留白过大或抽象概念需要可视化时使用。

---

## 何时需要配图/插图

- 内容页卡片不足导致留白 >120px：可添加轻量插图填充（优先调整卡片间距）
- 右侧留白 >250px：可用插图/图表填补
- 抽象概念/流程需辅助说明：用极简 SVG 矢量图
- **封面/总结页：不需要大面积装饰**。纯文字居中 + 大量留白即可。可选小号 emoji 或单色图标点缀

> 与 business 风格的关键差异：封面/总结不再强制"大面积装饰"，改为纯白/浅灰背景 + 文字居中 + 大量留白。

## 插图 SVG 排版规范

| 规则 | 要求 |
|------|------|
| 标签对齐 | text-anchor="middle"，标签 x = 图形 cx |
| 图元与标签间距 | 图形底部到标签基线 ≥ 22px |
| 辅助线到标签 | ≥ 20px |
| 线宽下限 | 可见描边 ≥ 1px |
| 插图内字号 | ≥ 13px（与 caption 一致） |
| 元素分布 | 均匀对称，视觉平衡；对比类插图以中轴线镜像对称 |
| 插图色彩 | 仅使用 chart_colors 序列色值，禁用渐变填充 |

## 封面/总结页留白规则

| 区域 | 要求 |
|------|------|
| 标题上方留白 | ≥ 120px |
| 标题与副标题间距 | ≥ 40px |
| 副标题下方留白 | ≥ 80px |
| 装饰 | 无。可选：标题上方 1 个 emoji（字号 48-56px）或单色小图标（28px, opacity 0.3） |
