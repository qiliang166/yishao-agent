# 2026-06-24 — PPT 预览改进 & 缩略图修复

## 改动

### 母版预览（HTML/CSS 形状渲染）
**问题**：文本预览只有文字，用户要求预览直接展示母版视觉效果（颜色、形状、背景）

**方案**：`/slides-content` 端点改为返回完整形状数据（位置、填充色、文本 run 的字体/大小/颜色/加粗/斜体），前端用绝对定位 HTML 元素渲染每张幻灯片。

**`backend/app.py`** — `get_template_slides_content` 重写：
- 遍历每个 slide 的所有 shape
- 提取 `left/top/width/height` 位置、`fill` 填充色、`rotation` 旋转
- 提取文本 run 的 `size/color/bold/italic/align`
- 提取 slide 背景色 `bg_color`

**`frontend/src/pages/TemplateManager.tsx`** — 预览面板重写：
- 用 `position:absolute` + `left/top/width/height %` 渲染形状
- 保留填充色、字体大小、颜色、加粗、斜体、对齐
- 缩略图条改为 mini 形状渲染（替代已损坏的 COM 图片）
- 母版背景色保留

### 缩略图修复
**问题**：卡片缩略图依赖 COM `previewSlides`（COM 在本机不可用），卡片无缩略图

**`backend/app.py`** — 新增 `GET /api/templates/{id}/slide-thumb`：
- 用 python-pptx + Pillow 从第 1 张幻灯片生成 PNG 缩略图
- 提取背景色 + 文本形状的相对位置渲染
- 缓存 1 小时（Cache-Control）

**`frontend/src/services/api.ts`** — 新增 `getSlideThumbUrl`

**`frontend/src/pages/TemplateManager.tsx`** — auto-thumbnail 改用 `slide-thumb` 端点

### 下载按钮移除
- 模板卡片上删除"下载"按钮，只保留"预览"
